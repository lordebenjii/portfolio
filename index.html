Flood Aid Kenya

FloodAidKE is a responsive web platform designed to help Kenyans report, monitor, and respond to flood emergencies in real time. It connects citizens, emergency responders, and government agencies through features like incident reporting with GPS, SOS requests, shelter location maps, and multilingual support. By streamlining communication and aid coordination, FloodAidKE empowers communities to act quickly and collaboratively during flood disasters.

HTML CSS Java Script JQuery

Maasai Mara Eco-Tourism

Masai Tourism Ventures is an immersive web experience showcasing authentic Maasai culture and sustainable tourism in Kenya. The platform connects travelers with guided cultural experiences, traditional ceremonies, and community-based eco-tourism activities, all led by local Maasai guides. With a focus on preserving heritage and supporting local livelihoods, it invites visitors to discover the beauty, resilience, and traditions of the Maasai people.
